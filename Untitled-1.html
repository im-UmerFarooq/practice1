GETTING STARTED

1.Downloading Git

First step you need to do is download Git, fot this go to download

2.Creating Account

The next step is to create you account for that go to github.com. Now click on create account to create, if you already have an account then just login.

3.Creating Repository

i) Now you need to create the repository in order to save you files, for this click on the +icon on the top right corner after you login, then click on "New repository".

ii) Now name your repository your repository and click on create repository below.

iii) After creating new repository your interface will look like this

4.Add Git to your Project Folder

i) For adding Git to you project folder go to your project folder and in the adress bar write "cmd" and press enter.

In CMD enter "git init" and press enter. You will see a folder named ".git" created in your project folder.
